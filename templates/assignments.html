As a beginner, each assignment this semester presented a challenge for me.

Although COVID-19 prevented us from holding classes, working on these projects at home has made me nostalgic.

From first assignment to last:

Homework 1a required us to understand binary numbers and Boolian Logic

Homework 1b was learning how to manipulate lists (i.e. reshuffle orders)

Homework 2a involved selection sort, sequential search, binary search, and calculating time complexity

Homework 2b was all about running simulations; the key here for me was naming variables correctly ;)

Homework 3a was doing set style operations on lists and solving recursively

Homework 3b demanded an understanding of numpy to solve a percolation problem in addition to reading from a dataset

The Midterm was changing images - this assignment required an understanding of PPM files

Homework 4 was using object-oriented codebase to correct a model and reading and interpreting error messages to fill in the missing details

Homework 5 was completing a machine learnign library and reading from a dataset in order to accomplish this

The Final is making our own webiste using HTML!

My favorite assignment this semester is this one because I feel like it encourages and rewards curiosity the most, finally something I'm good at! lol

But seriously, making a website for myself is something I've thought of doing before as a personal project, so being able to do it in class is super helpful/awesome.

Also, as someone who has never coded before, this assignment is fun because it bridges the gap between coding and creating.
